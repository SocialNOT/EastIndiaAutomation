# **App Name**: EIA Industrial AI Platform

## Core Features:

- 24/7 Multilingual Chat Agent: An AI-powered chat agent available round-the-clock in Bengali, Hindi, and English. Connects leads directly to the CRM.
- Real-Time Voice Agent: An ultra-low latency voice agent simulates human conversation, perfect for call handling and scheduling, via integration with the Vapi platform.
- Live Gemini Demo: Live demo environment to experience Gemini's text generation tool. This uses the VITE_GEMINI_API_KEY env variable, so protect your key.
- Live Vapi Demo: Voice channel interface to interact with the Vapi conversational agent tool in real-time. Utilizes VITE_VAPI_PUBLIC_KEY for voice processing and interaction.
- Data Ingestion & Integration: Secure infrastructure to ingest data from various sources and train bespoke enterprise LLMs. Guarantees data privacy.
- Automated Workflow Engine: Seamlessly connect and automate workflows between disconnected apps. Triggers actions across systems to streamline processes without intervention.
- Scrolling Ticker: Scrolling marquee highlighting key partnerships, tech features, and other performance metrics.

## Style Guidelines:

- Background: Deep Navy Blue (#0A192F) providing a sophisticated backdrop, shifting almost to black in certain areas to enhance depth.
- Accents: Brushed Bronze/Gold (#C9A227) for CTAs, borders, and interactive elements, imparting a touch of industrial elegance.
- Accent: Electric Cyan highlights used judiciously to indicate active AI states and processes, lending a subtle tech-forward glow.
- Headline font: 'Playfair', a heavy, authoritative Serif to convey stability and heritage.
- Body and data font: 'Inter' sans-serif font for clear readability of content.
- Bespoke vector icons in bronze, emphasizing machine/tech metaphors.
- Borders that dynamically "trace" with light on hover to provide a sense of tactile interaction.
